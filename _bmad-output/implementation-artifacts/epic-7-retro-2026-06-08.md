## O que foi bem

- **HDD construiu features reais do próprio orquestrador** (codegen, gate de testes) —
  one-shot, idiomático, CI verde. Arco "**HDD conserta o próprio loop**": F9
  descobre→conserta(10)→prova(11); gate(13)→bug `-uall`→auto-conserta(14).
- **Padrão oracle confiável:** validado RED/GREEN antes de enfileirar; pegou no-ops e
  forçou convergência. **Convergência sob oracle finalmente demonstrada ao vivo**
  (onda 9-t2: `->execute=4`, 3 correções via F2; prova do gate: `->execute=2` converge).
- **Provas ao vivo acharam o que os fakes esconderam** — o bug `-uall` do gate passou
  pelos unit tests (runner mockado) e só apareceu na prova ao vivo. *O valor do dogfood.*
- **Salvaguardas seguraram:** in-container, PR + gate humano, parada antes de quota/merge,
  sem `--admin`, prod intocada. Lição do #28/#33 aplicada (CI completo verde, Integração).

## O que não foi bem (sistêmico)

- **F7 t1:** PR no-op off-task passou o verify (sem oracle) e chegou ao gate — pego só
  pelo gate humano, não in-loop. (Gerou a generalização do gate.)
- **F9:** o agente hand-edita artefatos derivados (`openapi.json`) → onda escalou
  (desperdiçada) + salvamento manual no gate.
- **Gap híbrido de frontend** (`api-types.ts`) exigiu trabalho manual no gate até o
  codegen full-stack.
- **Erros de processo (Claude):** (a) criou `backup.sh` redundante sem checar o
  `scheduled-backup.sh`/runbook **pré-existentes** → reconciliação; (b) quase ativou o
  gate na prod (quebraria o smoke-test) antes de parar; (c) primeiras provas (F9, gate)
  inconclusivas, custaram iterações.

## Lições

1. **Provas ao vivo acham o que mocks escondem** — validar feature de loop end-to-end.
2. **Verify de feature precisa de oracle de aceitação** — DoD+drift prova "nada quebrou",
   não "a feature existe".
3. **Artefatos derivados são do sistema, não do agente** (codegen no loop).
4. **Checar artefatos/runbooks existentes antes de criar novos.**
5. **Contexto importa** — aplicar hardening cegamente na prod (smoke-test) pode quebrar
   o propósito dela; o classificador + a cautela pegaram o desvio.
